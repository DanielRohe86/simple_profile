Aprendi o básico sobre o Linux: comandos para o terminal que me permitem criar, remover ou mover arquivos entre pastas, modificar permissões, fazer downloads, instalar aplicativos e etc. Também foi possível aprender sobre estruturas de páginas web por meio dos recursos do HTML, depois complementado por conteúdos do HTML semântico, que permite a criação de páginas dentro de padrões internacionais. Estudei ainda os recursos do CSS, permitindo estilizar páginas web, tanto como elas se apresentam ao usuário, como também de forma interativa, por meio das pseudo-classes.
No momento em foi iniciado o estudo sobre o HTML, retomei algumas memórias de adolescência, quando construía sites para divulgar vídeos com shows de bandas. É como se eu estivesse seguindo uma trilha que eu já havia iniciado antes.
